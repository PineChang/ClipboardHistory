## 如何在 Xcode 中运行

1. 打开 Xcode → File → Open
2. 选择 `ClipboardHistory.xcodeproj`
3. 左上角选择目标 **My Mac**
4. 按 **Cmd+R** 运行
5. 菜单栏右上角出现 📋 图标，点击弹出面板
