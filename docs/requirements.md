# 需求文档

## 项目名称
**历史粘贴App** — macOS 剪贴板历史管理工具

## 用户画像
Mac 普通用户，需要频繁查找和使用历史复制内容，对电脑操作熟悉但不懂代码。

## 功能需求

### F1: 剪贴板监听
- 自动监听系统剪贴板变化
- 支持记录**文字**和**图片**两种内容类型
- 相同内容不重复记录（去重）

### F2: 历史记录浏览
- 以**卡片列表**形式展示所有历史记录
- 按**时间降序**排列（最新在最上面）
- 文字卡片显示内容摘要（前 100 字）
- 图片卡片显示缩略图
- 每条记录显示时间戳

### F3: 置顶功能
- 支持将任意条目**置顶**
- 置顶条目始终排列在列表最前面
- 可取消置顶

### F4: 删除功能
- 支持删除单条记录
- 删除操作有确认提示

### F5: 搜索功能
- 支持按文字内容搜索
- **实时过滤**，输入即搜
- 匹配文字内容（模糊包含匹配）

### F6: 再次粘贴
- 点击条目后，将内容恢复到系统剪贴板
- 自动粘贴到当前前台应用（模拟 Cmd+V）

### F7: 存储时长设置
- 可选：1 天 / 3 天 / 5 天
- 默认：3 天
- 过期条目自动清理

### F8: 开机自动启动
- 支持开机自动启动
- 可在设置中开启/关闭

### F9: 菜单栏驻留
- 软件常驻 macOS 菜单栏（右上角）
- 点击图标弹出历史面板
- 不占用 Dock 栏空间

## 非功能需求

### N1: 性能
- 剪贴板轮询间隔 0.5 秒，不造成系统卡顿
- 历史记录上限 500 条，超出后自动清理最旧记录
- 面板打开响应时间 < 200ms

### N2: 隐私
- 所有数据仅存储在本地
- 不上传任何数据到网络
- 图片存储在本地文件系统

### N3: 可用性
- 简洁直观，无需学习成本
- 支持 macOS 13 (Ventura) 及以上

## 不做的功能（V1 范围外）
- 多设备同步
- 富文本格式保留
- 文件复制记录
- 快捷键自定义
- 分类/标签
