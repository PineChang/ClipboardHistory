# 设计规范

## 设计理念
**简洁 · 直观 · 轻盈**

不喧宾夺主，安静地待在菜单栏，需要时快速调出，用完即走。

## 色彩系统

| 角色 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#F0F6FA` | 极浅蓝灰，大面积底色 |
| 卡片背景 | `#FFFFFF` | 白色卡片 |
| 卡片悬停 | `#E8F2F9` | 微蓝悬停态 |
| 主强调色 | `#5B9BD5` | 中蓝，按钮、图标、选中态 |
| 主强调浅 | `#A8D8EA` | 淡蓝，标签、次要装饰 |
| 文字主色 | `#1A1A2E` | 深蓝黑，正文 |
| 文字次要 | `#8E8E9A` | 灰，时间戳、摘要 |
| 分割线 | `#E8ECF1` | 浅灰蓝 |
| 危险色 | `#E05555` | 红，删除按钮 |

## 布局

### 面板
- 宽度: 340pt（固定）
- 高度: 520pt（固定）
- 圆角: 16pt
- 内边距: 12pt

### 搜索栏
- 高度: 36pt
- 固定在面板顶部
- 圆角: 10pt
- 背景: 白色，带淡蓝边框

### 卡片列表
- 可滚动区域，占据搜索栏和底部栏之间
- 卡片间距: 8pt

### 卡片
- 内边距: 12pt
- 圆角: 12pt
- 阴影: `y=2, blur=8, opacity=0.06`
- 悬停: 背景微蓝 + 轻微放大 (scale 1.01)
- 图片缩略图高度: 80pt，圆角 8pt

### 置顶标记
- 置顶卡片左侧有淡蓝色竖线指示器（宽 3pt）
- 置顶图标: SF Symbol `pin.fill`，淡蓝色

### 底部栏
- 高度: 36pt
- 设置按钮（齿轮图标） + 总条目数

## 图标

全部使用 Apple SF Symbols：
- 菜单栏: `list.clipboard`
- 置顶: `pin.fill` / `pin.slash.fill`
- 删除: `trash`
- 搜索: `magnifyingglass`
- 设置: `gearshape`
- 文字类型: `doc.text`
- 图片类型: `photo`

## 动效

- 面板打开: 淡入 + 轻微下移（200ms，ease-out）
- 卡片出现: 淡入 + 上移（150ms，ease-out，stagger 30ms）
- 卡片删除: 淡出 + 左移（200ms，ease-in）
- 卡片悬停: 背景色过渡（150ms）+ 缩放（150ms）
- 搜索结果切换: 淡入淡出（150ms）

## 状态

### 空状态
- 居中图标 `doc.text.magnifyingglass`
- 文案: "还没有复制记录" / "复制的内容会出现在这里"
- 副文案: "试试复制一段文字或图片吧"

### 搜索无结果
- 居中图标 `magnifyingglass`
- 文案: "没有找到相关内容"

### 加载状态
- 无（所有操作即时完成，无需加载态）
